Instrumentos de Evaluación Utilizados

Prueba Práctica de Programación

Ponderación: 25%

Descripción: Desarrollo de funciones y scripts en JavaScript para resolver problemas algorítmicos.

Criterios: Sintaxis correcta, lógica de programación, eficiencia del código y documentación.

Proyecto Web Integrador

Ponderación: 35%

Descripción: Creación de un sitio web completo con HTML, CSS y JavaScript que cumpla requisitos funcionales y estéticos.

Criterios: Diseño responsivo, funcionalidad, usabilidad, código limpio y presentación profesional.

Prueba Teórica

Ponderación: 20%

Descripción: Evaluación escrita sobre conceptos de programación web, protocolos HTTP, estructura DOM y buenas prácticas.

Criterios: Precisión conceptual, comprensión de estándares web y aplicación de conocimientos.

Evaluación Continua

Ponderación: 20%

Descripción: Valoración de tareas semanales, participación en clase, trabajo colaborativo y actitud profesional.

Criterios: Asistencia, puntualidad, responsabilidad, colaboración y ética profesional.
